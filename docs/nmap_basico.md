# Nmap — Guia básico e seguro

O Nmap é uma ferramenta de análise de rede usada para descobrir hosts e serviços.

⚠️ Só utilize em ambientes autorizados, como um laboratório.

## Exemplos educativos

Listar portas abertas:
nmap -sS 192.168.0.10

Identificar serviços:
nmap -sV 192.168.0.10

Varredura completa:
nmap -A 192.168.0.10

Use sempre com ética e responsabilidade.
